# LCD-USING-ARDUINO-NANO-

📟 Arduino Nano 16x2 LCD Display Interface (Proteus)

A simple Proteus simulation project demonstrating how to interface a 16x2 LCD (LM016L) with an Arduino Nano using 4-bit communication mode. The LCD displays a custom message on startup.

🚀 Project Overview

This project shows the basic interfacing of a 16x2 Character LCD with an Arduino Nano in Proteus.

The LCD displays:

ASHISH
PRAJAPAT

This project is useful for beginners learning:

Arduino LCD Interfacing
4-bit LCD Communication
Proteus Simulation
Embedded Systems Fundamentals
🛠 Components Used
Component	Quantity
Arduino Nano	1
16x2 LCD (LM016L)	1
Connecting Wires	As Required
Proteus 8 Professional	1
🔌 Circuit Connections
LCD Pin	Arduino Nano Pin
RS	D12
EN	D11
D4	D10
D5	D9
D6	D8
D7	D7
RW	GND
VSS	GND
VDD	+5V

LCD is operated in 4-bit mode to reduce the number of GPIO pins used.

▶️ How to Run
Open Proteus.
Load the project file.
Compile the Arduino sketch in Arduino IDE.
Generate the HEX file.
Attach the HEX file to Arduino Nano in Proteus.
Click Run Simulation.
Observe the text displayed on the LCD.
